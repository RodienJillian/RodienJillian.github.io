Crimson Bank Pro

The "Crimson Bank Pro" is a Python-based banking application built using Tkinter for the graphical user interface (GUI). It offers a user-friendly interface for managing basic banking operations such as creating bank account for savings, checking, debit/credit, allowing logging in and displaying bank details. The app features secure login functionality and easy navigation between different sections, ensuring a smooth user experience. Designed with simplicity and efficiency in mind, Crimson Bank Pro serves as an accessible tool for users to handle their financial tasks within a secure and intuitive environment.


Levenshtein Galore

Developed in collaboration with Sta. Ana, Eleina Anne.

"Lavenshteine Galore" is a dynamic application built with Python's Tkinter library for the graphical user interface and Python's powerful text processing capabilities for functionality. The application is centered around the concept of Levenshtein distance, a metric used to measure the difference between two strings by counting the minimum number of single-character edits (insertions, deletions, or substitutions) required to transform one string into another. The application offers two main features:

Homophone Edit Distance Game: This interactive game challenges users to calculate the Levenshtein distance between two words provided by the game. The words are homophones—words that sound the same but are spelled differently. This game tests and enhances the user's ability to identify and compute the edit distance, offering a fun and educational experience.

Minimum Edit Distance Measurer: This tool allows users to input any two words of their choice. The program then calculates and returns the minimum edit distance between these two words, helping users understand how similar or different the words are in terms of their spelling.

"Lavenshteine Galore" is an engaging way to explore the concept of edit distance, whether for educational purposes, self-improvement, or simply as a fun challenge.


Majestique App

Developed in collaboration with Sta. Ana, Eleina Anne.

Majestique App is an e-commerce application built with Python's Tkinter library for the graphical user interface and Python for functionality, designed specifically for accessory shopping. Upon launching the app, users can choose between two roles: personnel or customer.

Personnel Access: Personnel can log in using a secure passcode known only to them, granting access to manage inventory and view orders.

Customer Experience: Customers can browse the shop, learn about the app, place orders with real-time stock availability, add items to their cart, and proceed to checkout for a seamless shopping experience.
